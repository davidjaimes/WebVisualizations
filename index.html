Weather API

Collect temperature, humidity, cloud, and wind speed data from OpenWeatherMap API on 500+ randomly selected cities and use Matplotlib to visualize results.

Background: Create a Python script to visualize the weather of 500+ cities across the world of varying distance from the equator. To accomplish this, utilize a simple Python library, the OpenWeatherMap API, and a little common sense to create a representative model of weather across world cities.
